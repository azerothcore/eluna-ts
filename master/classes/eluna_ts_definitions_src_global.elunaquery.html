GetRow

  • GetRow(): number
    • Defined in eluna-ts-definitions/src/global.d.ts:1277

    Returns a table from the current row where keys are field names and values are the row's values. All numerical values will be numbers and everything else is returned as a string. For example, the query: SELECT entry, name FROM creature_template would result in a table like: { entry = 123, name = "some creature name" } To move to next row use [ElunaQuery:NextRow].

    Returns number

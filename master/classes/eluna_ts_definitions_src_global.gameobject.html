GetGUIDLow

  • GetGUIDLow(): number

  • Inherited from WorldObject.GetGUIDLow

    • Defined in eluna-ts-definitions/src/global.d.ts:2753

    Returns the low-part of the [Object]'s GUID. On TrinityCore all low GUIDs are different for all objects of the same type. For example creatures in instances are assigned new GUIDs when the Map is created. On MaNGOS and cMaNGOS low GUIDs are unique only on the same map. For example creatures in instances use the same low GUID assigned for that spawn in the database. This is why to identify a creature you have to know the instanceId and low GUID. See [Map:GetIntstanceId]

    Returns number
